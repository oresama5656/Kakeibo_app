# 📜 スキル：仕様策定 (write_specs)

`@pm` が最初に使用するスキルです。

## 📥 インプット
- ユーザーの要望（曖昧なものでも可）

## ⚙️ プロセス
1.  **ヒアリング**: 不明点があればユーザーに質問し、要件を具体化する。
2.  **仕様書作成**: `docs/spec.md` を作成し、機能一覧、UIデザイン案、データ構造を定義する。
3.  **計画作成**: `docs/plan.md` を作成し、スプリントごとの実装順序を定義する。
4.  **承認ゲート**: ユーザーに内容を提示し、**「承認（OK）」を得るまで次工程に進まない。**

## 📤 アウトプット
- `docs/spec.md`
- `docs/plan.md`
