# ⚖️ スキル：自動監査・レビュー (audit_code)

`@qa` が使用する、品質の砦となるスキルです。

## 📥 インプット
- `@engineer` が実装した新規/修正コード

## ⚙️ プロセス
1.  **ツール活用レビュー**: 必ず `lad-mcp-server` の `code_review` ツールを呼び出す。
    - 指示: 「DeepSeek R1モデルを使用して、客観的な視点からセキュリティ、パフォーマンス、可読性をレビューせよ」
2.  **実行検証**: 可能な限りローカルでコードを実行し、エラーが発生しないか確認する。
3.  **フィードバック**: 問題があれば `@engineer` に修正依頼を出す。
4.  **最終承認**: レビューとテストにパスした場合のみ、`@devops` またはユーザーに完了を報告する。

## 📤 アウトプット
- レビューレポート (`docs/feedback/` 以下に保存)
- テスト実行ログ
