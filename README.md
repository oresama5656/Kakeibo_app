# 📊 シンプル家計簿 (Kakeibo Web App)

シンプルで、Googleスプレッドシートとリアルタイムに同期できる家計簿Webアプリです。
PCとスマートフォンの両方から、広告なし・完全無料で資産管理ができます。

## 🌟 主な特徴

- **リアルタイム・クラウド同期**: Google Sheets APIを使用して、あなたのGoogleドライブ内のスプレッドシートにデータを直接保存。
- **ひと目でわかる資産推移**: Chart.jsを使用したダッシュボードで、資産の増減をグラフで可視化。
- **爆速入力UI**: スマホ片手でも入力しやすい大きなボタンと、カテゴリー/口座を絵文字で選べるアイコンピッカー。
- **銀行CSVのスマート入力**: 銀行からDLしたCSVを無加工で読み込み。見出しのゆれを自動吸収し、入出金を即座に判別。
- **ダークモード対応**: システム設定や好みに合わせて、目に優しい表示に切り替え可能。
- **安心のプライバシー**: サーバーを持たない純粋なWebアプリ。データはあなたのGoogleアカウント内にのみ保存されます。

## 🛡️ 安心・安全へのこだわり (Googleスプレッドシート連携の強み)

このアプリは、一般的な家計簿アプリとは異なり「データの所有権をユーザーに持たせること」を最優先に設計されています。

- **データ所有権はユーザー自身にある**: 開発者のサーバーにデータが一切保存されないため、サービスが突然終了してもデータが消える心配がありません。
- **高い透明性**: 何が記録されているか、いつでも自分のスプレッドシートを開いて、生のデータを直接確認・検証できます。
- **二次利用が簡単**: 記録されたデータはただのスプレッドシートなので、標準機能で自由にグラフを作ったり、Excel形式でダウンロードして確定申告や詳細な分析に活用したりできます。

## 📥 銀行CSVのスマート一括入力

CSVファイルを使用して、大量の取引データを一括で登録できます。銀行からダウンロードしたCSVを加工せずにそのまま読み込める「スマート検出機能」を搭載しています。

- **柔軟な見出し検知**: 
    「日付」の代わりに「取引日」や「年月日」、「摘要」の代わりに「お取引内容」など、銀行ごとの微妙な表記の違いを自動で解釈します。
- **自動種類判定 (入出金2列対応)**: 
    「お引出し」列に金額があれば自動で **支出** 、「お預入れ」列に金額があれば自動で **収入** として読み込みます。CSV側に「種類」列を自作する必要はありません。
- **安心のプレビュー機能**: 
    読み込んだデータは即座に保存されず、一旦「一括入力テーブル（下書き）」に展開されます。内容を確認し、カテゴリーなどを修正してから「一括登録」ボタンを押すことで、家計簿に反映されます。
- **既存テンプレートも対応**: 
    従来の `日付,種類,カテゴリー,金額...` 形式の専用テンプレートも、引き続きそのまま読み込み可能です。


## 🚀 セットアップガイド

個人のGoogleアカウントを使用して、自分専用の家計簿をデプロイ・利用するための手順です。

### 1. Google Cloud Console での準備
1. [Google Cloud Console](https://console.cloud.google.com/) にアクセスし、プロジェクトを作成します。
2. **APIとサービス** > **ライブラリ** から以下の2つを検索し、**有効化** します。
   - `Google Sheets API`
   - `Google Drive API`
3. **OAuth 同意画面** を設定します（「外部」を選択し、テストユーザーに自分のメールアドレスを追加）。
4. **認証情報** > **認証情報を作成** > **OAuth クライアント ID** を作成します。
   - **アプリケーションの種類**: ウェブ アプリケーション
   - **承認済みの JavaScript オリジン**:
     - `http://localhost:3000` （ローカル開発用）
     - `https://[あなたのGitHubユーザー名].github.io` （GitHub Pages公開用）
   - **クライアント ID** をコピーしておきます。

### 2. プログラムの構成
1. `js/auth.js` を開き、先ほど取得したクライアントIDを `CLIENT_ID` 変数に貼り付けます。
2. リポジトリを GitHub にプッシュし、**GitHub Pages** で公開設定を行います。

## 📂 フォルダ構成
- `index.html`: アプリのエントリポイント。
- `css/`: デザインシステム（ダークモード・アニメーション）。
- `js/`: アプリケーションロジック。
  - `app.js`: アプリ全体の初期化とルーティング。
  - `auth.js`: Google OAuth2 / SDK 連携。
  - `sync.js`: WebSocketによるリアルタイム更新通知。
  - `store/`: データ管理（Redux風のストア構成）。
    - `index.js`: ストアの統括窓口（Coordinator）。
    - `SyncManager.js`: クラウド同期エンジンの核心。
    - `TransactionStore.js`: 取引データの操作。
- `docs/`: 仕様書・計画書・レポート類。
- `worker/`: Cloudflare Workers (バックエンド) ソース。

## 🔧 使用技術
- **Frontend**: Vanilla JavaScript (ES Modules)
- **Backend**: Cloudflare Workers (WebSocket / KV)
- **Database**: Google Sheets (Cloud-first)
- **Charts**: Chart.js
- **Icons**: Lucide Icons
