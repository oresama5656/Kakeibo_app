# 📋 家計簿Webアプリ 要件定義書

> **ステータス**: 確定（v1.0）
> **最終更新**: 2026-03-30
> **設計方針**: シンプルイズベスト

---

## 1. 目的・コンセプト

広告なし・完全無料で、PCとスマホからリアルタイムに同期して使える**家計簿Webアプリ**。

### コアバリュー
1. **アイコンタップによる直感的な入力** — プルダウンは使わない
2. **「貯まってる実感」の可視化** — 総資産推移グラフ
3. **シンプルな構成** — 余計な機能を作らない、バグを減らす

---

## 2. システム構成

| フロントエンド | HTML / CSS / JavaScript | レスポンシブ（PC・スマホ両対応） |

## 5. 画面構成

### 5.1 📝 入力画面（メイン・最優先）

**UIの気持ちよさが最重要。**

```
┌─────────────────────────────┐
│ [支出] [収入] [振替]        │  ← 種類切替ボタン
├─────────────────────────────┤
│  ⚡ショートカット            │  ← よく使うセット（横スクロール）
│  [コンビニ] [スーパー] ...  │
├─────────────────────────────┤
│        ¥ 1,280              │  ← 金額表示
├─────────────────────────────┤
│  口座を選択                  │
│  ┌──┐ ┌──┐ ┌──┐            │
│  │💴│ │👛│ │🏦│ ...        │  ← pinned が先頭
│  │現金│|小遣|│常陽│           │
│  └──┘ └──┘ └──┘            │
│  ▼ もっと見る               │  ← 残り11件を展開
├─────────────────────────────┤
│  カテゴリーを選択            │
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐      │
│  │🍚│ │🛒│ │🍻│ │💰│      │  ← pinned が先頭
│  │食料│|日用|│交際│|小遣│     │
│  └──┘ └──┘ └──┘ └──┘      │
│  ▼ もっと見る               │  ← 残りを展開
├─────────────────────────────┤
│  📅 今日  │ 📝 メモ(任意)    │
├─────────────────────────────┤
│  7  8  9                     │
│  4  5  6                     │  ← テンキー
│  1  2  3                     │
│  C  0  ✓                    │  ← ✓ で登録
└─────────────────────────────┘
```

#### UI/UXのこだわりポイント
- **pinned項目を上部固定**: 口座・カテゴリーともに、よく使うものが先頭に表示
- **折りたたみ**: pinned以外は「もっと見る」で展開（画面が埋まらない）
- **グリッドサイズ**: スマホで押しやすい最低 48x48px、余白を十分に確保
- **選択時フィードバック**: タップするとアイコンがバウンスアニメーション＋色変化
- **テンキーは常に画面下部に固定**（スマホ片手操作に最適化）

### 5.2 📊 ダッシュボード

**シンプルに2つ:**

1. **総資産推移グラフ**（折れ線）
   - 期間切替: 1ヶ月 / 3ヶ月 / 6ヶ月 / 1年 / 全期間
   - 「貯まってる実感」を与える

2. **口座別残高カード**
   - 各口座の現在残高をアイコン付きカードで一覧表示
   - 総資産をヘッダーに大きく表示

### 5.3 📋 履歴画面

- 取引一覧（日付の新しい順）
- フィルター: 期間、口座
- 各行をタップ → 編集 or 削除

### 5.4 ⚙️ 設定画面

- 口座の管理（追加・編集・削除・pinned切替・並び替え）
- カテゴリーの管理（追加・編集・削除・pinned切替・並び替え）
- ショートカットの管理（追加・編集・削除）
- Google Sheets 接続設定（スプレッドシートID）
- ダークモード切替

## 6. UI/UX 要件

### デザイン方針
- **モバイルファースト** — スマホ片手操作に最適化
- **PC**: サイドバーナビ + コンテンツの2カラム
- **ダークモード**: システム設定に追従 + 手動切替
- **余計な装飾を排除**: 情報が即座に伝わるクリーンなUI

### カラー
- 収入: 緑系 (#4CAF50)
- 支出: オレンジ/赤系 (#FF6B6B)
- 振替: 青系 (#42A5F5)
- 背景: ダーク (#1a1a2e) / ライト (#f8f9fa)

### グリッドサイズ（アイコン選択UI）
- 最小タップターゲット: **48×48px**（Googleのガイドライン準拠）
- アイコン間余白: **8px以上**
- スマホ横幅: 4列グリッド（アイコン+ラベル）
- PC横幅: 6-8列グリッド

## 7. ロジック要件

### 残高計算
```
口座残高 = 初期残高
          + Σ(その口座への収入の amount)
          - Σ(その口座からの支出の amount)
          + Σ(その口座への振替の amount)
          - Σ(その口座からの振替の amount)

総資産 = Σ(全口座の現在残高)
```

### 資産推移グラフ
- 全取引を日付順にソート → 各日の総資産を累積計算
- スナップショット不要（取引データから都度算出）
