# ダークモード完全対応・仕様書 (spec.md)

## 概要
プレミアム・リデザインされたすべてのUI（履歴、分析）において、テーマ（ライト/ダーク）切り替えに完全対応させる。

## 具体的ルール
1.  **カラーの抽象化**: 全ての「特定の青色」や「特定の文字色」をCSS変数化し、直接色コードを指定することを禁止する。
2.  **ダークモードの質感**: 単に黒くするのではなく、ダークブルーや深いネイビーを基調とした「プレミアム・ダーク」を実現する。
3.  **チャートの適応**: 背景色が変わってもチャートが見やすいよう、透明度や枠線の太さを調整する。

## 適応箇所
- 履歴画面 (History Screen)
- 分析画面 (Analysis Screen / PL & BS)
- 新しく追加したカード UI すべて
