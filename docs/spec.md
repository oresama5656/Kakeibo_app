# 分析画面・リデザイン仕様書 (spec.md) - シニアデザイナー監修

## 1. デザインビジョン
「Appleの引き算」×「日本の整理術」。
30〜40代の大人が信頼を寄せる、無駄のない美しさと圧倒的な情報の読み取りやすさを両立する。

## 2. 具体的なデザイン改善（分析タブ）

### A. ヘッダーの整理（デクラッタリング）
- **現状の課題**: 週/月/年、支出/収入、チャート切り替えのボタンが散乱し、情報の主役（金額）を邪魔している。
- **改善案**: 
  - 最上部に「深縹（ふか縹）」を基調としたグラデーション背景の「サマリー・マスター・カード」を配置。
  - 金額を最大級のフォントで表示し、Apple純正アプリのような重厚感を出す。
  - 各種切り替え（週/月/年など）は、タブ形式で1箇所に集約し、選択時のみ強調する。

### B. 配色と質感
- **配色**: 伝統的な和の色「深縹（ふか縹 / #2a4073）」や「エンジ（#7a2622）」をアクセントに使用。
- **背景**: 純白ではなく、極めて薄いグレー（#f8f9fa）で、プレミアムな質感を出す。
- **ブラウザ臭の払拭**: 標準のボタン枠やドロップダウンを廃止し、余白と影（Soft Shadow）で要素を定義。

### C. タイポグラフィ
- 金額: 大きく、力強く。`font-weight: 900`。
- カテゴリ名: 控えめながらも視認性を確保。
- 日付期間: キャプションとして小さく、しかし読みやすい位置に配置。

### D. チャートの洗練
- ドーナツチャートの配色は、彩度を抑えた「大人向けカラーパレット」に統一。
- 中央に代表的な情報を表示する。

## 3. ユーザー体験（UX）
- 操作を少なく。最も見たい情報を中心に配置。
- ドリルダウン（詳細表示）への動線をスムースに。
