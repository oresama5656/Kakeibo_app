# 履歴画面ヘッダー・デザイン仕様書 (spec.md) - 日本向けプレミアム・エディション

## 1. デザインコンセプト
「清潔感、信頼感、情報の整理」。
日本の家計簿アプリに多く見られる、視認性の高いカードスタイルを採用しつつ、Fintechらしい洗練された配色で高級感を演出する。

## 2. 具体的なデザイン改善点（新案）

### A. カードスタイル・タイル
- フィルター要素（期間、口座、カテゴリ）を、背景色付きの角丸カード（Tile）の中に配置。
- 画像にあるような、ブルーグレーやダークグリーン系の落ち着いたトーンをベースにする。

### B. フラット＆クリーンなタイポグラフィ
- 日本語フォント（Noto Sans JP）を活かし、ウェイト（太さ）の使い分けで情報の重要度を表現。
- 日付や金額の数字は、可読性の高いサンセリフ体を使用。

### C. ヘッダー情報の整理
- 1行目に現在の日付や資産状況のサマリーを表示。
- その下に、カード形式で「期間選択」「口座フィルター」「カテゴリフィルター」を並べる。

### D. 配色設計
- **ベースカラー**: 淡いグレー (#f5f6fa)
- **アクセント**: ブルーグレー (#4a5568), ダークグリーン (#1a365d) などの「深みのある色」を使用。
- **テキスト**: 漆黒ではなく、少しグレーを混ぜた深い紺色で目に優しく。

## 3. 実装上の工夫
- スマホで見切れないよう、カードの幅を自動調整しつつ、必要に応じてカルーセル（横スクロール）を検討。
- 各カードには薄いシャドウを加え、立体感を出す。
