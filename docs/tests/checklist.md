# クリティカル機能・回帰テストチェックリスト (Regression Checklist)

この文書は、Evaluator エージェントがリリース前に必ず確認すべき項目を定義する。

## 1. 起動とナビゲーション (Stability)
- [ ] URL `http://localhost:3000` でエラーなく起動する
- [ ] コンソールに `Uncaught SyntaxError` や `404 (Not Found)` が出ていない
- [ ] 下部ナビゲーションの5つのボタンがすべて機能する (Input, Dashboard, History, Analysis, Settings)

## 2. 分析画面 (Analysis - refactored in Sprint 2)
- [ ] 「収支」タブでドーナツチャートが表示されている
- [ ] 「資産」タブで棒グラフと折れ線グラフが表示されている
- [ ] 期間切り替え（週・月・年）をクリックして、表示金額が更新される

## 3. 設定画面と並び替え (Settings & Sortable)
- [ ] 口座一覧が表示されている
- [ ] 口座の左側にドラッグハンドル `⠿` が存在している
- [ ] (要確認) 口座を並び替えた後、リロードしても順序が維持されている

## 4. ピン留め機能 (Pinned Status)
- [ ] 特定のカテゴリ/口座をピン留めできる
- [ ] 保存/読込（あるいは同期）を行っても、ピン留めされた項目がリストの上部に表示され続ける

## 5. クラウド同期 (Cloud Sync)
- [ ] Google 認証ボタンが表示されている
- [ ] 同期実行時に `pinned` フィールドが失われない (Fixed in Sprint 2)

---
**判定基準:**
- 上記 Must 項目 (1, 2, 3) のいずれかが欠けている場合は **FAIL** とする。
