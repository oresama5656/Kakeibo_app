# 分析機能強化（PL/BS導入）実装プラン

## ステップ1：ダッシュボードの整理
- [ ] ダッシュボードから「資産推移」グラフを削除
- [ ] ダッシュボードから「口座別残高推移」グラフを削除
- [ ] ダッシュボードを「総資産額」と「各口座カード」に絞り、スッキリさせる

## ステップ2：分析タブへの集約とモード切替の実装
- [ ] 分析タブ上部に「収支 (PL)」と「資産 (BS)」の切替ボタンを配置
- [ ] ダッシュボードから移動したグラフを「資産 (BS)」モードに統合
- [ ] グラフ形式（円グラフ / 折れ線グラフ）の切替が、資産モードでも機能するように調整

## ステップ3：収支 (PL) モードの強化
- [ ] 現在の「カテゴリー別内訳（円グラフ）」に加え、「月ごとの収支（棒グラフ/折れ線グラフ）」を追加
- [ ] 月別の収支合計（収入 - 支出）を計算・表示するロジックを実装
- [ ] モチベーション維持のための「貯蓄率」バッジの表示

## ステップ4：資産 (BS) モードの強化
- [ ] 各口座の金額比率を示す「資産構成（円グラフ）」を実装
- [ ] 過去の資産推移と、特定の口座に絞った推移をスムーズに見れるようにUIを調整
- [ ] 指定した時点の「純資産（資産 - 負債）」を算出・表示

## ステップ5：UI/UXのブラッシュアップ
- [ ] グラフ上のデータポイントをクリックした際に、その月の詳細（内訳）へジャンプする機能を検討
- [ ] モバイル画面でのチャートの視認性最適化
